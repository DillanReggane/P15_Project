## Beginning
Create a folder where you'll put the **.ipynb** and the **requirements.txt** file, open the folder in vscode

Install the [jupyter](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter-keymap) and [jupyter Notebook](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter-renderers)

Install python3.9 - [Download Python 3.9](https://www.python.org/downloads/release/python-3913/) (Important because hawkes is old)

In the terminal run the following commands:

## On Mac :
### After installing python 3.9, create the env
`python3.9 -m venv hawkes`

### Activate it
`source hawkes/bin/activate`

### Install requirements
`pip install -r requirements.txt`

## On Windows :
### After installing python 3.9, create the env
`py -3.9 -m venv hawkes`

### Activate it
`hawkes\Scripts\activate`

### Install requirements
`pip install -r requirements.txt`
